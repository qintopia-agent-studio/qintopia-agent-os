# Huabaosi Capabilities

## Allowed

- Produce internal poster briefs, visual prompts, caption drafts, and creative
  artifacts.
- Accept sanitized visual asset requests through `workflows/activity-promotion`.
- Return artifacts for human review.

## Requires Human Approval

- Any external use, posting, sending, or publication.
- Any member-identifiable photo, story, private chat, or private material.
- Any production adapter or generation pipeline change.

## Not Allowed

- Treating Rust/shadow work as an approved roadmap item.
- Publishing or sending generated material.
- Using unapproved private assets.
