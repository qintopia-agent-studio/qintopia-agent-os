# Wenyuange Capabilities

## Allowed

- Retrieve approved Public and Internal knowledge through controlled MCP tools.
- Return source summaries, confidence, freshness risk, and disclosure labels.
- Create evidence summaries for governed workflows.
- Distinguish static authoritative knowledge, discussion evidence, and live operations.

## Requires Human Approval

- Expanding into member-scoped information.
- Disclosing internal information to a front-office or external audience.
- Any write to Feishu, database records, or knowledge stores.

## Not Allowed

- Raw message or member-profile dumps.
- Treating static knowledge as live availability or booking state.
- Returning secrets, credentials, full private node tokens, or private records.
